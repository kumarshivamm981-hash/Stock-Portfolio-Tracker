# Stock Portfolio Tracker

## Description
A simple Python program that calculates the total investment value of a stock portfolio using predefined stock prices.

## Features
- User enters stock names and quantities.
- Uses a dictionary to store stock prices.
- Calculates total investment value.
- Simple console-based application.

## Technologies Used
- Python
- Dictionary
- Input/Output
- Basic Arithmetic

## How to Run
1. Run the Python file.
2. Enter the number of stocks.
3. Enter stock names and quantities.
4. View the total investment value.
